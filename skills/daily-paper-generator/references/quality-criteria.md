# Paper Quality Evaluation Criteria — Social Science Edition

## Dimensions & Weights

| Dimension | Weight | Core Question |
|-----------|--------|---------------|
| **Contribution & Novelty** | 30% | Does it advance knowledge in a meaningful way? |
| **Identification Strategy** | 25% | Is the causal/empirical claim credible? |
| **Relevance to Research Areas** | 20% | Fits comp politics / poleco / public econ / econ history / text-as-data? |
| **Data & Method Quality** | 15% | Valid measurement, adequate sample, robustness? |
| **Writing & Clarity** | 10% | Clear argument, well-structured, transparent? |

Score each dimension 1–5 → weighted sum → select top 3.

---

## Detailed Scoring Rubrics

### 1. Contribution & Novelty (30%)

| Score | Standard |
|-------|----------|
| **5** | Resolves a long-standing debate, introduces a new paradigm, or provides first evidence on an important question |
| **4** | Significant new finding or mechanism; clearly advances the literature |
| **3** | Useful contribution with incremental advance; adds to accumulation of evidence |
| **2** | Modest improvement on existing work; limited novelty |
| **1** | Replication or marginal tweak; minimal contribution |

**Key questions:**
- Does it answer a question the field has been asking?
- Does it open a new research agenda?
- Is the contribution clearly stated relative to the prior literature?

---

### 2. Identification Strategy (25%)

*For empirical papers (causal inference / quasi-experimental):*

| Score | Standard |
|-------|----------|
| **5** | Clean natural experiment, credible instrument, or RCT; threats to validity fully addressed |
| **4** | Strong design (RDD, DID, IV) with good robustness checks; minor concerns addressed |
| **3** | Reasonable design with acknowledged limitations; main findings likely hold |
| **2** | Weak identification; alternative explanations not ruled out |
| **1** | Purely correlational with causal language; no credible identification |

*For theory papers:*

| Score | Standard |
|-------|----------|
| **5** | Elegant formal model with clear applied relevance; proofs complete |
| **4** | Well-specified model, propositions proven, generates testable predictions |
| **3** | Reasonable model, main results proven, some assumptions restrictive |
| **2** | Ad hoc assumptions, limited generalizability |
| **1** | Model poorly motivated or results trivial |

*For text-as-data / NLP papers:*

| Score | Standard |
|-------|----------|
| **5** | Human validation of automated measures; links to substantive outcome |
| **4** | Validation against gold standard, multiple robustness checks |
| **3** | Some validation, reasonable measure construction |
| **2** | Weak validation; construct validity concerns |
| **1** | No validation; measure of unclear meaning |

---

### 3. Relevance to Research Areas (20%)

Core research areas to assess against:
- Comparative politics (institutions, regime type, parties, elections)
- Political economy (redistribution, fiscal policy, political connections)
- Political institutions (state capacity, legislative/executive politics)
- Public economics (taxation, social insurance, public goods)
- Economic history (long-run development, historical institutions)
- Causal inference methodology
- Text as data / NLP for social science
- Applied game theory

| Score | Standard |
|-------|----------|
| **5** | Directly in core research area; immediately citable |
| **4** | Close neighbor; highly relevant methodology or finding |
| **3** | Broadly relevant; useful for context or method |
| **2** | Tangentially related; minor relevance |
| **1** | Outside scope |

---

### 4. Data & Method Quality (15%)

| Score | Standard |
|-------|----------|
| **5** | Novel high-quality dataset; comprehensive robustness; replication code available |
| **4** | Strong data with clear documentation; multiple robustness checks |
| **3** | Adequate data; standard robustness checks; some limitations acknowledged |
| **2** | Limited sample or measurement issues; few robustness checks |
| **1** | Poor data quality or measurement; no robustness |

**Key questions:**
- Is the sample large enough / representative?
- Are key variables well-measured?
- Are standard robustness checks included (placebo, alternative specs)?
- Is replication data/code promised or available?

---

### 5. Writing & Clarity (10%)

| Score | Standard |
|-------|----------|
| **5** | Exceptionally clear; argument flows logically; contribution unmistakable |
| **4** | Clear and well-organized; contribution clearly stated |
| **3** | Readable; main argument understandable |
| **2** | Some confusion; argument hard to follow in places |
| **1** | Poorly written; difficult to assess merit |

---

## Composite Score Calculation

```
Total = Contribution×0.30 + Identification×0.25 + Relevance×0.20 + Data/Method×0.15 + Writing×0.10
```

**Example:**
- Contribution: 4 → 4×0.30 = 1.20
- Identification: 4 → 4×0.25 = 1.00
- Relevance: 5 → 5×0.20 = 1.00
- Data/Method: 3 → 3×0.15 = 0.45
- Writing: 4 → 4×0.10 = 0.40
- **Total: 4.05 / 5.00**

---

## Screening Workflow

1. **Title + abstract scan** — exclude clearly irrelevant papers
2. **Source bonus** — NBER WP from PE/POL program or top-5 journal = +0.2 boost
3. **Full evaluation** — score all 5 dimensions for remaining candidates
4. **Rank** — sort by total score
5. **Select top 3** — write bilingual summaries
